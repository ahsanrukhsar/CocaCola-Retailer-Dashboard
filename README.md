# CocaCola-Retailer-Dashboard
CocaCola Retailer Dashoboard on MS Excel


This dashboard provides an all-in-one solution for tracking sales performance, managing inventory, and gaining valuable customer insights. Some of the key highlights include:

**Sales Insights**: A detailed breakdown of sales by beverage brand and retailer, showcasing year-over-year growth and variances.

**Profit Analysis**: Visualized quarterly sales and operating profit margins to help pinpoint trends and optimize business strategies.

**Interactive Filters**: Easily filter data by region and year to drill down into specific insights and make data-driven decisions.

This project has been an incredible experience, enhancing my skills in data visualization and Excel dashboard creation. Excited to continue leveraging data to drive impactful business results!
